## Abstract Factory 抽象工廠

> 當產品具有多組相同性質的分類(品牌與系列)，可以用抽象工廠減少工廠的數量
> 抽象工廠又被稱為「工廠的工廠」

## 品牌與系列範例
- A品牌汽車工廠
  - 轎車生產線
  - 越野車生產線
  - 跑車生產線

- B品牌汽車工廠
  - 轎車生產線
  - 越野車生產線
  - 跑車生產線
